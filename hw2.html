Q1.

  1. Calculate the extinction probability for the following branching processes (starting with one individual at time \(0\)):
    1. the process with a Binomial\((2,2/3)\) offspring distribution;
    2. the process with offspring distribution given by \(p_j=2^{-(j+1)}\) for \(j\ge 0\);
    3. the process with generating function \(\varphi(s)=s\).
  2. Calculate the function \(\varphi_n(s)=\mathbb{E}[s^{Z_n}]\) for general \(n\) for the offspring distribution in (ii).

Q2. We call an event \(A\) for a branching proces \((Z_n)_{n \ge 0}\) "inherited" if

  • \(\{\)extinction\(\} = E \subset A\); and
  • whenever \(A\) occurs, it must also occur for each of the sub-processes starting from each individual in generation \(1\).
For example, \(E\) itself is "inherited" because if the process goes extinct, so must each subprocess starting from each individual in generation \(1\).
  1. Is the event \(\{\lim_{n \to \infty} Z_n/a^n = 0\}\) inherited for \(a > 0\)? What about \(\{\lim_{n\to\infty} Z_n/n^n = 0\}\)?
  2. Show that if \(A\) is inherited then \(\phi(\mathbb{P}(A)) \ge \mathbb{P}(A)\), and deduce that \(\mathbb{P}(A)\) must be equal to \(\rho\) or \(1\).
